## Project Overview

Commercial-Cleaning is a static marketing site for a commercial cleaning service, made up of a home page and an "About Us" page, styled with Bootstrap and custom CSS.

## Features

- **Multi-Page Layout:** Separate home (`index.html`) and about (`about-us.html`) pages.
- **Bootstrap Components:** Responsive grid and UI components from Bootstrap.
- **Icon Support:** Font Awesome icons used throughout the site.

## Tech Stack

- **Frontend:** HTML5, CSS3, Bootstrap
- **Icons:** Font Awesome

## Folder Structure

```bash
Commercial-Cleaning/
├── css/              # Stylesheets (Bootstrap + custom)
├── font-awesome/      # Font Awesome icon library
├── fonts/             # Custom web fonts
├── js/                # JavaScript used by Bootstrap/UI components
├── media/             # Media assets used across the site
├── index.html         # Home page markup
├── about-us.html       # About Us page markup
└── style.css          # Custom page styling
```

## Installation Guide

### Prerequisites

- A modern web browser
- (Optional) A local static server such as VS Code's "Live Server" extension

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/rajesh-ranjan-git/Commercial-Cleaning.git
   cd Commercial-Cleaning
   ```

2. Open `index.html` directly in your browser, or serve the folder with a local static server:

   ```bash
   npx serve .
   ```

## Usage Instructions

1. Open `index.html` in your browser to view the home page.
2. Navigate to the "About Us" page to view company details.
